# Rental Bond Tracker

## 📌 Overview
Rental Bond Tracker is a full-stack web application designed to help tenants in Australia manage rental bond payments and tenancy records in one centralised platform.

## 🚨 Problem Statement
Tenants lack a centralised system to manage:
- Rental bond payments  
- Lease agreements  
- Inspection records  
- Communication with landlords  

This leads to:
- Lost documents  
- Disputes over bond refunds  
- Poor organisation of tenancy history  

---

## 💡 Proposed Solution
A web-based system that allows tenants to:
- Track bond payments and status  
- Store tenancy documents  
- Record inspections with evidence  
- Maintain a clear rental timeline
